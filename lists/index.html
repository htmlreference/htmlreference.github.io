  3. ????
  4. PROFIT!!!

# ol

Defines an ordered list.

Example: Copy

  1. Step one
  2. Step two
  3. ????
  4. PROFIT!!!

type

Defines how the list is numbered.

"1"

Default.

Uses numbers.

  1. One
  2. Two
  3. Three
  4. Four
  5. Five

"a"

Uses lowercase letters.

  1. One
  2. Two
  3. Three
  4. Four
  5. Five

"A"

Uses uppercase letters.

  1. One
  2. Two
  3. Three
  4. Four
  5. Five

"i"

Uses lowercase Roman numerals.

  1. One
  2. Two
  3. Three
  4. Four
  5. Five

"I"

Uses uppercase Roman numerals.

  1. One
  2. Two
  3. Three
  4. Four
  5. Five

start

Defines a number to start the list with.

"3"

You can use any integer.

  1. One
  2. Two
  3. Three
  4. Four
  5. Five

reversed

Reverses the order of the list.

No value required.
  1. One
  2. Two
  3. Three
  4. Four
  5. Five
Type: block Self-closing: No Permalink Share MDN
